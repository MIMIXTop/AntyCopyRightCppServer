# AntyCopyRightCppServer

Простой HTTPS-сервер на C++ для обработки запросов, связанных с образовательными курсами и студентами.

## Описание

Этот проект представляет собой асинхронный HTTPS-сервер, построенный с использованием Boost.Beast и Boost.Asio. Сервер обрабатывает различные типы запросов, такие как получение списка курсов, работ, студентов и загрузка заданий.

## Функции

- Асинхронная обработка HTTP/HTTPS запросов
- Поддержка SSL/TLS шифрования
- Обработка запросов для курсов, студентов и заданий
- Конфигурируемый через config.hpp

## Требования

- C++23
- CMake 3.20+
- Boost (с компонентами json, stacktrace_addr2line, url, headers)
- OpenSSL
- Google Test (для тестов)

## Сборка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd AntyCopyRightCppServer
   ```

2. Создайте директорию сборки:
   ```bash
   mkdir build
   cd build
   ```

3. Сконфигурируйте проект с CMake:
   ```bash
   cmake ..
   ```

4. Соберите проект:
   ```bash
   make
   ```

## Запуск

После сборки запустите сервер:
```bash
./AntyCopyRightCppServer
```

Сервер будет слушать на `127.0.0.1:8080` с использованием SSL-сертификатов `../server.crt` и `../server.key`.

## Тестирование

Для запуска тестов:
```bash
cd build
ctest
```

Или напрямую:
```bash
./test/RunAllTests
```

## Структура проекта

- `src/` - исходный код сервера
- `test/` - модульные тесты
- `build/` - директория сборки
- `CMakeLists.txt` - конфигурация CMake
- `config.hpp.in` - шаблон конфигурации
